Настройки sec-lib

Все настройки библиотеки находятся в классе SecConfig.php.

  • $_secBaseDir

    Абсолютный путь к sec-lib библиотеке. Устанавливается по умолчанию как

                                            
      
                                            
    1. $this->_secBaseDir = dirname(__FILE__) . DIRECTORY_SEPARATOR;
      2.
    Внимание!Путь всегда должен заканчиваться косой чертой (/).
  • $_secSessionName

    Имя сессии. По умолчанию PHPSESS.

  • $_secSecureCookies

    По умолчанию: 

                                            
      
                                            
    1.  public $_secSecureCookies = 1;
      2.

    Если установлена в 1, то устанавливается и проверяется абсолютное время жизни кук.

  • $_secSecureSession

    По умолчанию: 

                                            
      
                                            
    1.  public $_secSecureSession = 1;
      2.

    Если установлено в 1, то устанавливается и проверяется абсолютное время жизни сессии.

  • $_secOnerrorRedirectTo

    Этому свойству присваивается ссылка, куда пользователь будет перенаправлен в случае: ошибки безопасности, обнаружении XSS атак, невалидных токенах.

  • $_secSessionHeadersCheck

    По умолчанию: 

                                            
      
                                            
    1.  public $_secSessionHeadersCheck = 1;
      2.

    Если установлено в 1, сессия привязываеится к используемому браузеру. Изменение браузера повлечет за собой очистку кук и сессии.

  • $_secLog

    По умолчанию: 

                                            
      
                                            
    1.  public $_secLog = 1;
      2.

    Если установлено в 1, то сообщения об ошибках, обнаруженные атаки и неправильнык токены будут записаны в файл var/log.txt.

  • $_secSessionLifeTime

    По умолчанию: 

                                            
      
                                            
    1.  public $_secSessionLifeTime = 7200;
      2.

    Время жизни сессии.

  • $_secSessionAbsoluteLifeTime

    По умолчанию: 

                                            
      
                                            
    1.  public $_secSessionAbsoluteLifeTime = 21600;
      2.

    Абсолютное время жизни сессии.

  • $_secTokenLifeTime

    По умолчанию: 

                                            
      
                                            
    1.  public $_secTokenLifeTime = 7200;
      2.

    Время жизни токенов.

  • $_secSessionRefresh

    По умолчанию: 

                                            
      
                                            
    1.  public $_secSessionRefresh = 7200;
      2.

    Время, через которое будет перезаписана сессия.

  • $_secIdsOnAttackAction

    По умолчанию: 

                                            
      
                                            
    1.  public $_secIdsOnAttackAction = 'logout redirect';
      2.

    Реация приложения на найденные атаки и неправильные токены.
