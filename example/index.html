PHP Security Library

Это небольшая библиотека на PHP, которая поможет повысить уровень безопасности вашего web-приложения. С ее помощью вы можете предотвратить следующие выды атак:

  • XSS (Cross Site Scripting)
  • SQL-Injection
  • CSRF (Cross Site Request Forgery)
  • Session-Fixation
  • Mail-Header-Injection
  • HTTP-Header-Manipulation
  • HTML code injection

Все методы и свойства библиотеки статические, что упрощает ее использование.

Установка

Скопируйте директорию sec-lib в корневую папку вашего web-приложения. Затем подключите класс Security.php c помощью php функций require_once или include_once и вызовите статический метод run(). 

                                
    
                                    
  1. require_once 'sec_lib/Security.php';
    2. 
                                    
  2. Security::run();
    3.

Внимание! Подключение библиотеки и вызов метода run() следует выполнять после инициализации сессии.

Внимание! У папки var, в корне библиотеки, должны быть права на запись, иначе файлы логов не будут созданы.
